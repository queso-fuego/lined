# lined

Line Editor, clone or homage to _THE_ standard text editor

## dependencies
- C compiler with -std=c23 support (Clang 19 or 20+, GCC 14 or 15+, etc.).
See [here](https://en.cppreference.com/w/c/23.html) for an overview.
- make with ::= support

## building
`make`

## usage
`./lined <file>`
